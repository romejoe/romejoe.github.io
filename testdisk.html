    Welcome to the page.

    1. Boot from LiveCD and navigate to desktop
    2. Open terminal and run 
      sudo apt-get update 
sudo apt-get upgrade
sudo apt-get install testdisk
      This installs TestDisk.
    3. Run the following command: 
      sudo testdisk
    4. Following this, I would follow this drive, looking at the following guides:
      • https://www.cgsecurity.org/wiki/TestDisk_Step_By_Step#Disk_selection
        Note:
        • You will probably have an EFI partition table.
      • http://selfsolve.net/software/testdisk.html
